Normalice CSS - Unificar estilo de todos los navegadores

Los navegadores tienen un conjunto predeterminado de estilos CSS que utilizan para representar los elementos. Estos estilos son distintos en cada navegador por lo que una misma página puede verse y comportarse de forma diferente dependiendo del navegador utilizado.

El fichero normalize.css anula las inconsistencias más comunes entre los estilos de los navegadores y corrige errores conocidos. Esto permite mantener los estilos similares en los navegadores y que la web se vea igual independientemente del navegador utilizado

¿Qué hace?

  • Preserva los valores predetermiados por defecto de los navegadores a diferencia de algunos CSS reset
  • Normaliza estilos para un amplio rango de navegadores
  • Corrige errores e inconsistencias de los navegadores
  • Mejora la usabilidad con modificaciones sutiles

¿Cómo utilizarlo?

  1. Descargar el fichero de la página oficial y colocarlo en la carpeta de los css del proyecto web
  2. Crear la referencia en el <head> del documento html: <link href="css/normalize.css" rel="stylesheet" type="text/css">

Insertar videos de Youtube

Para insertar videos de Youtube se utiliza el código de inserción que proporciona Youtube en la opción compartir de cada video. Éste código no utiliza la etiqueta <video> sino que utiliza <iframe>

Uso básico de la etiqueta video HTML5

Para insertar un video del que disponemos la ruta completa podemos utlizar la etiqueta <video> en la que se especifica la ruta del video en el atributo src.

Otros atributos básicos del video son width y height para especificar el ancho y alto del video respectivamente. El atributo controls muestra los controles para que el usuario pueda controlar el vídeo.

Control de video por voz

El boton ">" permite iniciar y parar la reproducción del video de forma manual.

Pulsar el botón "Comenzar reconocimiento voz" para que el navegador comience a escuchar. Dar permiso al uso del micrófono si se solicita. En la etiqueta de la derecha se mostrará el mensaje "Escuchando" cuando el reconocedor de voz esté activo.

Decir:

  • "Reproducir": para que se inicie la reproducción del video.
  • "Parar: para detener la reproducción del video.
